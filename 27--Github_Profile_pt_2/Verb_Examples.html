//3 examples of each verb being used in the Github API

1. GET:Used for retrieving resources.
i.e curl https://api.github.com/users/mema82

2. POST:Used for creating resources, or performing custom actions (such as merging a pull request)
i.e curl --user "mema82" --request POST --data

3. PUT:Used for replacing resources or collections. For PUT requests with no body attribute, be sure to set the Content-Length header to zero.
i.e curl --user "mema82" -X POST --data '{"description":"Created via API","public":"true","files":{"file1.txt":{"content":"Demo"}}' https://api.github.com/gists
